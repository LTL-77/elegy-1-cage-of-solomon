# Contributing

This project is still in early development, so contributions should stay tightly aligned with the current demo slice.

## Before Opening A PR

- read `README.md`
- read the files under `docs/`
- keep changes inside the current demo scope unless a maintainer explicitly expands it

## Good Contribution Targets

- bug fixes
- UI clarity improvements
- battle readability improvements
- tooling and project structure improvements
- typo fixes in public-facing documentation

## Changes To Avoid Without Prior Discussion

- major story rewrites
- scope expansion beyond the current vertical slice
- replacing the combat direction
- introducing large third-party dependencies
- adding redistributable assets without clear licensing

## Asset And Lore Boundary

This repository intentionally excludes private setting archives and unreleased narrative material. Do not submit private or unlicensed materials into the public repo.

# Demo Vertical Slice

## Goal

Build a short playable demo that proves the game's tone, narrative structure, and combat identity without implementing the full production scope.

## Target Runtime

- first internal playable: `15-25` minutes
- polished public demo target: `20-40` minutes

## Slice Flow

1. Title screen
2. New game
3. Opening dialogue sequence
4. Static scene interaction in Beiyao
5. Tutorial combat
6. Short narrative transition
7. Boss preparation scene
8. Boss combat against `Solomon Beiyao`
9. Resolution scene
10. Save / return to title

## Required Systems

- game flow manager
- dialogue presentation
- scene interaction nodes
- battle bootstrap
- turn order and action resolution
- simple battle result handoff
- save/load scaffolding

## Explicitly Out Of Scope For Slice 1

- multi-route chapter structure
- full six-character party system
- complete relationship system
- full equipment crafting economy
- NG+ / second playthrough logic
- Android release work
- production-grade art pipeline
